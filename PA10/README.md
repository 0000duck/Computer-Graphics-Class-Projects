
# PA10

### Build &amp; Run

    mkdir build
    cd build
    cp ../makefile .
    make
    ./PA10

### Dependencies
- SDL2
- SDL2_Mixer
- GLEW
- GLM
- Assimp
- Bullet
- stb_image (source provided)
- stb_truetype (source provided)
- Dear ImGui (source provided)
	- uses stb_truetype, stb_textedit, stb_rect_pack
