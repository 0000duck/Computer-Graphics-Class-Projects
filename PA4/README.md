# PA4: Spinning Cube

## Libraries
GLM
GLEW
SDL

## Shaders
shaders are located in the folder titled "shaders"
Vertex Shader - VertexShader.vert
Fragment Shader - FragmentShader.frag

## Building
$ cd build
$ make

## Running
from build folder:
$ ./PA4 -v ../shaders/VertexShader.vert -f ../shaders/FragmentShader.frag -m ../models/object.obj
